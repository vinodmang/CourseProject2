This file contains some description of the included files and the steps to run the R code to ouput a tidy dataset from the raw data.

Description of provided files : 
codebook.md : This file contains the background details of the data, the processing steps implemented in the R code to produce the tidy dataset from the raw data.

tidydata.txt : This file contains the tidy dataset and is the result of executing the provided R code (run_analysis.R)

run_analysis.R : contains the R code to read in the raw data and output the tidy dataset called "tidydata.txt" into the current working directory.

To run the code, open the R application and set the working directory to point to the raw data folder with command (for example setwd("C:/RLearning/data/UCI HAR Dataset"). Drag and drop the run_analysis.R file into the R console
